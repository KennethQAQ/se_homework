# **Java插入排序的性能分析**  
## 插入排序代码：  
    public static void sort(int[] arr)
    {
        int i, j;
        int n = arr.length;
        int target;

        for (i = 1; i < n; i++)
        {
            j = i;
            target = arr[i];

            while (j > 0 && target < arr[j - 1])
            {
                arr[j] = arr[j - 1];
                j--;
            }

            arr[j] = target;
        }
    }
